7‑Zip — бесплатный архиватор с высокими показателями сжатия

Формат 7z с алгоритмом LZMA/LZMA2, открытая лицензия, консольная утилита 7z.exe.

Краткий обзор

7‑Zip — свободно распространяемый архиватор, известный эффективностью и поддержкой множества форматов.

Поддерживаемые форматы

  • Создание: 7z, ZIP, TAR, WIM
  • Распаковка: RAR/RAR5, ARJ, CAB, GZ, ISO, XZ, Z и др.

Ключевые возможности

  • Высокая степень сжатия (LZMA/LZMA2)
  • Шифрование AES‑256 (7z)
  • Интеграция в контекстное меню Проводника
  • Портабельная версия, локализация интерфейса

Как пользоваться

Через Проводник

  1. Выделите файлы/папки → ПКМ → «7‑Zip» → «Добавить к архиву…».
  2. Выберите формат 7z или ZIP, метод сжатия, пароль (опционально).
  3. Подтвердите и дождитесь завершения.

Через командную строку

  1. Используйте 7z.exe (в каталоге установки) или добавьте в PATH.
  2. Примеры: 
    7z a -t7z -m0=lzma2 -mx=9 backup.7z .\data
7z a -t7z -pPAROL -mhe=on -v500m archive.7z .\big_folder
7z t archive.7z
7z x archive.7z -o.\restore
-mx=9 — макс. сжатие; -mhe — шифрование имён; -v — тома.

Рекомендации

  • Для лучшего сжатия используйте 7z/LZMA2 и словарь побольше (при достаточной ОЗУ).
  • Для совместимости выбирайте ZIP.
  • Для больших архивов используйте тома: -v100m, -v1g и т.д.

Дополнительно

Лицензия и исходники

  • Свободно распространяется под лицензией LGPL; часть модулей — под собственными лицензиями
  • Исходный код доступен на официальном сайте 7‑Zip
  • Разрешено использование в коммерческих целях при соблюдении условий лицензии
  • Активное сообщество разработчиков и переводчиков

Системные требования

  • Windows 7/8/10/11 (x86/x64); портативная версия доступна
  • Низкие требования к ОЗУ/CPU; для больших словарей LZMA2 нужно больше ОЗУ
  • Поддержка локализаций интерфейса, интеграция в Проводник
  • Версии для Linux и macOS (p7zip)

Особенности формата 7z

  • Методы LZMA/LZMA2, PPMd, BZip2; словарь до сотен МБ
  • Шифрование AES‑256, опция шифрования имён (‑mhe)
  • Высокая степень сжатия на медленных пресетах (mx=9) ценой ��ремени
  • Поддержка твёрдых архивов для лучшего сжатия

Шпаргалка по CLI

:: Создать архив 7z с максимальным сжатием
7z a -t7z -m0=lzma2 -mx=9 -mfb=64 -md=256m backup.7z .\data

:: Создать зашифрованный архив с томами по 500 МБ
7z a -t7z -pPAROL -mhe=on -v500m archive.7z .\big_folder

:: Проверить/распаковать
7z t archive.7z
7z x archive.7z -o.\restore

Практические рекомендации

  • Для обмена с разными системами используйте ZIP; для максимального сжатия — 7z/LZMA2
  • Подбирайте размер словаря исходя из доступной ОЗУ; тестируйте mx=7/9
  • Для бо��ьших наборов данных используйте тома (‑v) и проверку целостности
  • Регулярно обновляйте 7‑Zip для поддержки новых форматов
