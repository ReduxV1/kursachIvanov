Выбор файловой системы и разметка диска в Windows 11

Практическое руководство по выбору NTFS/exFAT и разметке дисков с помощью встроенных инструментов Windows 11.

Какую файловую систему выбрать

Кратко о назначении распространённых ФС в Windows 11.

NTFS (рекомендуется для системного и внутренних дисков)

  • Поддержка прав доступа (ACL), шифрования (EFS), сжатия, квот
  • Надёжность: журналирование, самовосстановление
  • Ограничения: совместимость с другими ОС ниже, чем у exFAT
Используйте NTFS для системного диска и постоянных внутренних разделов.

exFAT (съёмные носители, обмен данными)

  • Совместима с Windows, macOS и многими устройствами
  • Нет ограничений 4 ГБ на файл (в сравнении с FAT32)
  • Нет журналирования и развитой безопасности как в NTFS
Выбирайте exFAT для флешек и внешних SSD/HDD, если важна совместимость.

FAT32 (устаревшая, для совместимости со старыми устройствами)

  • Макс. размер файла 4 ГБ, раздела — до 32 ГБ в стандартных утилитах
  • Минимум возможностей по безопасности и надёжности
Используйте только при необходимости совместимости со старыми устройствами.

ReFS (серверные сценарии)

  • Повышенная устойчивость к повреждениям, масштабируемость
  • Ограниченная поддержка в клиентских версиях Windows 11
Для домашних/учебных сценариев обычно не требуется.

Разметка диска: базовый план

Пример разумной схемы для одного физического диска в ПК.

Системный раздел (C:)

  • Файловая система: NTFS
  • Размер: 80–150 ГБ (в зависимости от объёма ПО)
  • Назначение: Windows 11, программы

Данные (D:)

  • Файловая система: NTFS
  • Размер: остальное пространство диска
  • Назначение: документы, проекты, учебные материалы

Внешние носители

  • exFAT для флешек и переносных SSD/HDD
  • Для совместимости между Windows/macOS/устройствами
Перед любыми операциями с разделами сделайте резервную копию важных данных.

Инструменты Windows 11

Два встроенных способа управлять разделами и файловыми системами.

Графически: Управление дисками

  1. Нажмите Win+X → «Управление дисками».
  2. Выберите диск/раздел → ПКМ → «Сжать том…» или «Удалить том…» (при необходимости).
  3. На нераспределённой области: ПКМ → «Создать простой том…» → укажите размер и букву.
  4. Выберите файловую систему: NTFS (для внутренних), exFAT (для внешних).
  5. Быстрое форматирование и метка тома → Готово.

Консольно: diskpart (для опытных пользователей)

  1. Откройте «Командная строка (Администратор)» или PowerShell (Администратор).
  2. Запустите diskpart и выберите диск/раздел: 
    diskpart
list disk
select disk 0
list partition
select partition 3
  3. Создание раздела и форматирование: 
    create partition primary size=102400
format fs=ntfs quick label=Data
assign letter=D
exit
Будьте внимательны: команды diskpart необратимы при ошибочном выборе диска/раздела.

Схема разметки диска: MBR или GPT

Что выбрать и как проверить текущую схему.

GPT (GUID Partition Table)

  • Рекомендуется для Windows 11 с UEFI
  • До 128 разделов без расширенных/логических
  • Поддержка больших дисков (>2 ТБ)
  • Повышенная надёжность (резервные таблицы, CRC)
Выбирайте GPT на современных ПК с прошивкой UEFI — это требование для установки Windows 11.

MBR (Master Boot Record)

  • Совместимость со старым BIOS
  • Ограничение диска ~2 ТБ и до 4 первичных разделов
  • Требует расширенных/логических разделов при >4
Используется для старых систем. Для Windows 11 предпочтителен GPT.

Как проверить схему (MBR/GPT)

  1. Win+X → «Управление дисками» → ПКМ по диску (слева) → «Свойства» → вкладка «Тома» → «Стиль разделов».
  2. Через PowerShell (Администратор): 
    Get-Disk | Format-List Number, FriendlyName, PartitionStyle, Size

Как конвертировать MBR → GPT

Без потери данных: mbr2gpt (для системного диска)

  1. Создайте резервную копию важных данных.
  2. Откройте «Командная строка (Администратор)».
  3. Проверьте диск: 
    mbr2gpt /validate /disk:0 /allowFullOS
  4. Выполните конвертацию: 
    mbr2gpt /convert /disk:0 /allowFullOS
  5. В BIOS/UEFI включите режим UEFI (выключите Legacy/CSM) и перезагрузитесь.
Требуются свободные слоты для системных разделов EFI/MSR. Утилита выдаст причину отказа при проверке.

С очисткой диска: diskpart (удаляет данные)

  1. Сделайте резервную копию. Действие необратимо.
  2. Откройте Командную строку (Администратор): 
    diskpart
list disk
select disk 1
clean
convert gpt
exit
  3. Создайте разделы и форматируйте (см. раздел «Инструменты Windows 11»).
Этот метод стирает все разделы и данные выбранного диска.

Рекомендации и замечания

  • Для системного и рабочих разделов внутри ПК — NTFS.
  • Для обмена с другими ОС/устройствами — exFAT на внешних носителях.
  • Резервное копирование данных перед изменением разметки обязательно.
  • Проверяйте тома: chkdsk C: /scan или через свойства диска.
  • Используйте «Сжать том» вместо удаления, чтобы не потерять данные при переразметке.
