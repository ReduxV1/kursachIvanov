AOMEI — решение для резервного копирования

Плановые бэкапы, инкрементальные копии, локальные и сетевые хранилища.

Краткий обзор

Инструмент для автоматизации резервного копирования рабочих данных в Windows.

Поддержка задач

  • Полные, инкрементальные, дифференциальные копии
  • Шифрование и сжатие архивов резервных копий
  • Планировщик и уведомления о статусе

Назначения

  • Локальные диски и сетевые папки (SMB)
  • Внешние накопители

Как настроить

Быстрый старт

  1. Создайте новую задачу резервного копирования.
  2. Выберите источники (папки/файлы) и место назначения.
  3. Выберите тип копии (полная/инкрементальная) и расписание.
  4. Включите шифрование/сжатие при необходимости и сохраните.

Практические советы

  • Проверяйте восстановление на тестовом каталоге раз в месяц.
  • Храните хотя бы 3 последние полные копии.

Дополнительно

Системные требования

  • Windows 7/8/10/11; Windows Server 2008 R2–2022
  • ОЗУ: от 2 ГБ; Свободное место под кэш/логи: от 200 МБ
  • Права администратора для установки и драйверов VSS/снимков
  • Поддержка UEFI и Legacy BIOS для системных копий

Редакции и лицензирование

  • Standard/Professional — рабочие станции и домашние ПК
  • Server/Technician/Technician Plus — серверы и обслуживание
  • Trial — пробный период с ограничениями функционала
  • Корпоративные лицензии с централизованным управлением

Функциональные возможности

  • Горячие копии через VSS, исключения/фильтры по маскам
  • Схемы ротации: FIFO, по возрасту/числу версий, по размеру
  • Уведомления: email/лог/статусы; отчёты по расписанию
  • Клонирование дисков и миграция на SSD

Интеграции и хранилища

  • SMB/NAS/iSCSI, внешние диски, поддержка облаков через подключенные диски
  • Pre/Post‑скрипты для подготовки сервисов и валидации результатов
  • Экспорт/импорт задач для переноса конфигурации
  • Поддержка различных файловых систем: NTFS, FAT32, exFAT

Практики эксплуатации

  • Правило 3‑2‑1, периодические тесты восстановления (Disaster Recovery Day)
  • Раз в N инкрементов выполняйте полную копию для ограничения «цепочки»
  • Храните конфигурацию задач под контролем версий/резервируйте её
  • Мониторьте свободное место на целевых дисках

Частые проблемы и решения

  • VSS Error — перезапустите службу VSS, проверьте тени и место на диске
  • Access denied — запустите от администратора, проверьте ACL/блокировки
  • Медленный бэкап — исключите временные/кеш‑папки, проверьте антивирус
  • Ошибки клонирования — проверьте целевой диск на ошибки
