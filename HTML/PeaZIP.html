PeaZip — бесплатный кроссплатформенный архиватор

UI-оболочка для ряда движков сжатия, богатая поддержка форматов, портативные сборки.

Краткий обзор

PeaZip — свободный архиватор с удобным интерфейсом, поддержкой множества форматов и кроссплатформенностью.

Поддерживаемые форматы

  • Создание: 7z, ZIP, TAR, WIM, PEA
  • Распаковка: RAR/RAR5, ACE, ARJ, CAB, GZ, ISO, XZ и др.

Ключевые возможности

  • Поддержка шифрования (AES‑256 и др.)
  • Разделение на тома, проверка целостности
  • Интеграция в контекстное меню Проводника
  • Портабельные версии без установки

Как пользоваться

Через интерфейс

  1. Откройте PeaZip → «Add» (Добавить) для создания архива.
  2. Выберите формат (например, 7z/zip), метод сжатия и пароль (опционально).
  3. Нажмите «OK» и дождитесь завершения.

CLI (при наличии консольных инструментов)

  1. PeaZip использует внешние движки (7z/pea и др.). Команды аналогичны 7‑Zip.
  2. Пример (7z): 
    7z a -tzip backup.zip .\data
7z x backup.zip -o.\restore
Для некоторых форматов потребуется установить соответствующие backend‑утилиты.

Рекомендации

  • Для лучшего сжатия используйте форматы 7z/LZMA2.
  • Для совместимости выбирайте ZIP.
  • Проверяйте архивы после создания средствами проверки целостности.
